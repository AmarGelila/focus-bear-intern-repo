# Setting up a React project #32

### Done using vite and yarn

### Steps :-

1. Run 'yarn create vite project-name --template react' command
2. Agree to install yarn and start the project
3. Run 'yarn add tailwindcss @tailwindcss/vite' command to use tailwindcss in the project
4. Configure tailwindcss in vite.config.js file
